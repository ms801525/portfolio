Quote Of The Times

Quote of the Times is a lightweight, responsive, and fast-loading static website built with HTML, CSS, and JavaScript. It dynamically displays a famous quote every minute, creating an engaging experience for users seeking daily inspiration. The website is designed for simplicity and performance, ensuring quick load times even on slower connections. New quotes are added regularly to keep the content fresh and encourage repeat visits. The clean UI and minimalist design make it easy for users to focus on the quotes themselves.

Services Used: S3, CloudFront, SSL, SquareSpace, Cloudwatch.

For this project, I designed and implemented a fully automated CI/CD pipeline using GitHub Actions to deploy a static website (quote of the times) to AWS S3. The workflow is triggered on every push to the main branch, automatically syncing files with the S3 bucket, invalidating the CloudFront cache, and deploying updates live without manual intervention. I also configured a custom domain purchased via Squarespace, managing DNS records to point to AWS resources. This setup ensures a seamless deployment process and a production-ready environment where changes are reflected instantly. By leveraging GitHub Actions, I was able to create an efficient and scalable workflow that aligns with modern DevOps practices.

Services Used: GitHub Actions, AWS S3, Cloudfront, SquareSpace DNS.
