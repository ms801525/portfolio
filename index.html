Quote Of The Times

Built Quote Of The Times using HTML, CSS, and JavaScript, it is a statis website that generates random quotes from the click of a button Deployed a fully secure static website using AWS S3, CloudFront, and SSL via ACM with DNS validation, achieving HTTPS with no public file exposure Implemented Origin Access Control (OAC) to restrict direct S3 access and configured CloudFront for content delivery, caching, and domain routing via Cloudflare

The website quoteofthetimes.info is a minimalist tool designed to provide users with a daily dose of inspiration. Upon visiting, users are greeted with a simple interface featuring a prominent "Get Quote" button. Clicking this button generates a famous quote, which users can then copy to their clipboard for easy sharing or personal reflection.
